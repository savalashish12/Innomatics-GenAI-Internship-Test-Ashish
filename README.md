Food Delivery Data Analysis

This project was completed as part of the Innomatics Research Labs – GenAI Internship Entrance Test.
It focuses on combining data from different sources and analyzing it to generate business insights.

Files

users.json – User details

restaurants.sql – Restaurant data

analysis.py – Data processing & analysis

final_food_delivery_dataset.csv – Final merged dataset

Tools

Python, Pandas, SQLite3, JSON, Jupyter Notebook

I have Done

Loaded data from CSV, JSON, and SQL

Merged all datasets using left joins

Analyzed revenue, AOV, seasonality, and user behavior
